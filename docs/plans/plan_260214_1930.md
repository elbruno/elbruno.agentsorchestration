# Plan: Radical Dashboard UI Overhaul

**Date:** 2026-02-14 19:30  
**Status:** In Progress

## Summary

Overhaul the Blazor Server dashboard with a modern dark-themed design, SVG agent graph with animated connections, console-style activity feed, collapsible panels, and chat-based prompt memory for iterative orchestration runs.

---

## Changes

### 1. Visual Style Overhaul (`wwwroot/app.css`)

- Dark theme with CSS custom properties (background `#0d1117`, cards `#161b22`, borders `#30363d`)
- Accent colors per agent role matching `AgentConfiguration.Color`
- Smooth transitions on all interactive elements
- Monospace font for the console feed, system font stack elsewhere
- Pulse keyframe animation for active agent nodes
- Glow effect on active SVG connection lines

### 2. Agent Graph with SVG Connections (`AgentGraph.razor`)

- Hub-and-spoke SVG layout: Orchestrator centered, Planner/Coder/Designer around it
- Animated SVG path lines from Orchestrator to each agent
- Lines glow/pulse when `AgentConnection.IsActive` is true
- Nodes show icon, name, model, and truncated instruction
- Status ring: gray idle, pulsing blue active, green complete, red error
- Responsive viewBox sizing

### 3. Console-Style Activity Feed (`ActivityFeed.razor`)

- Move from right sidebar to bottom panel (full width)
- Dark background, monospace font, green-on-black console aesthetic
- Each line prefixed with timestamp and agent icon/color
- Auto-scroll to bottom (latest item) using JS interop `scrollTop = scrollHeight`
- Show last 200 items (up from 50)

### 4. Collapsible Panels

- All panels (Agent Graph, Activity Feed, Workspace Viewer) wrapped in a collapsible container
- Click header to toggle collapse with smooth CSS transition
- Chevron indicator rotates on collapse/expand
- **Workspace Viewer collapsed by default**, others expanded

### 5. Chat-Based Prompt Memory (`PromptInput.razor`, `Home.razor`)

- Replace single textarea with a chat conversation view
- `List<ChatMessage>` tracks user prompts and orchestration summaries
- Previous messages displayed above the input as a scrollable chat log
- On each run, the full conversation context is passed to agents so they can build on prior output
- Agents reuse the same workspace directory across conversation turns
- Clear conversation button resets history and workspace

### 6. Dashboard Models (`DashboardModels.cs`)

- Add `Color` and `Icon` to `AgentNodeState`
- Add `ChatMessage` record (Role, Content, Timestamp)
- Add `AgentConnection` class (From, To, IsActive)

### 7. Home.razor Layout Restructure

- Full-width layout (no more 8/4 column split)
- Top: Chat input with message history
- Middle: Agent Graph (full width)
- Bottom-left: Workspace Viewer (collapsible, collapsed by default)
- Bottom: Activity Feed console (collapsible, expanded)

### 8. Getting Started Guide (`docs/getting-started.md`)

- Add more complex multi-step prompts demonstrating iterative conversation
- Add a "Multi-Turn Conversation" example showing follow-up prompts
