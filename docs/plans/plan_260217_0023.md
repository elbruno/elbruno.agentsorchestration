# Plan: Dynamic Agent Loading from Awesome Copilot Repository

**Date:** 2026-02-17 00:23  
**Status:** ✅ Completed  
**Issue:** Dynamic agent loading feature

## Overview

Implement a complete feature to list and download agent definitions from the Awesome Copilot Repository. Once agents are retrieved, they should be added to the orchestration, and the orchestrator and planner should be aware of these agents for use in code generation tasks.

## Requirements

### Core Functionality
1. ✅ Create a library to fetch agent definitions from GitHub
2. ✅ Parse `.agent.md` files with YAML front matter
3. ✅ Register dynamic agents alongside static agents
4. ✅ Cache downloaded agents locally
5. ✅ Support loading from multiple sources (GitHub, URL, file)

### Integration Requirements
1. ✅ Integrate with existing `AgentConfigurationStore`
2. ✅ Support both static (enum) and dynamic agents
3. ✅ Provide DI extensions for easy setup
4. ✅ Maintain backward compatibility

### Sample Scenarios
1. ✅ **Scenario 1 - WinForms Expert**: Load WinForms expert to validate WinForms Designer-compatible code
2. ✅ **Scenario 2 - Security Reviewer**: Load Security Reviewer to review plans and implementations

## Technical Decisions

### Agent Identification
- Used synthetic `AgentRole` enum values for dynamic agents (hash-based)
- Maintains compatibility with existing `IAgentClient` interface
- Allows seamless switching between static and dynamic agents

### Caching Strategy
- Local file cache with configurable expiration (default 24 hours)
- Cache keys based on agent ID
- Automatic cache directory creation

### Parser Implementation
- Regex-based YAML front matter extraction
- Fallback to full content if no front matter
- Support for quoted values and arrays in YAML

### Icon/Color Assignment
- Deterministic hash-based assignment for consistency
- Special icons for common agent types (security, testing, etc.)
- Default icon for unrecognized types

## Usage Example

```csharp
// Initialize
var staticStore = new AgentConfigurationStore();
var loader = new AwesomeCopilotAgentLoader();
var manager = new DynamicAgentManager(staticStore, loader);

// Load specialized agents
await manager.LoadAndRegisterAgentAsync("WinFormsExpert");
await manager.LoadAndRegisterAgentAsync("se-security-reviewer");

// Query agents
var allAgents = manager.GetAllAgents(); // Returns 13 agents
var winForms = manager.GetByName("WinForms Expert");
bool hasAgent = manager.HasAgent("SE: Security");
```
