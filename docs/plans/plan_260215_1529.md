## Phase 2: Persistence — Save & Load Sessions (MEDIUM PRIORITY)

### 2.1 Session Persistence Service

**Scope:** Save/load sessions to/from JSON with workspace isolation

**File:** `src/ElBruno.AgentsOrchestration.Core/Orchestration/SessionPersistence.cs` (NEW)

**Responsibilities:**

- Serialize `ConversationSession` to JSON
- Deserialize JSON back to `ConversationSession`
- Determine persistence path (per-workspace or global session store)
- Handle file I/O with error handling

**Design:**

```csharp
public sealed class SessionPersistence
{
    public string PersistenceRootPath { get; }
    
    public Task SaveSessionAsync(ConversationSession session, string filePath);
    public Task<ConversationSession?> LoadSessionAsync(string filePath);
    public Task<IReadOnlyList<ConversationSession>> ListSavedSessionsAsync();
}
```

**Effort:** ~3 hours | **Value:** High (enables session resumption, multi-device sync)

## Phase 3: Context Management — Smart History Handling (MEDIUM PRIORITY)

### 3.1 Context Size Awareness

**Scope:** Prevent context from exceeding token limits

**File:** `src/ElBruno.AgentsOrchestration.Orchestration/Orchestration/Models.cs` (extend `ConversationSession`)

**Add to `ConversationSession`:**

```csharp
public int EstimateTokenCount() { /* rough estimate */ }
public ConversationSession TrimHistory(int maxTokens) { /* remove oldest turns */ }
public ConversationSession TrimToLastNTurns(int n) { /* keep only last N */ }
```

**Design:**

- Simple token estimation: ~1 token per 4 characters
- Let callers decide when to trim (no automatic trimming)
- Preserve first turn always (context anchor)

**Effort:** ~1.5 hours | **Value:** Medium (prevents token overflow with long conversations)
